# Supplementary Materials for “Transformation to structured perturbations for a Fiedler linearization”

This repository contains three complementary materials that support the numerical example presented in our manuscript (LAA-D-25-00839). All files are directly linked to the illustrative example in Section 7 of the paper.

## 1. MATLAB code – `verssalALPcodematlab.m`

A complete MATLAB implementation of the algorithm described in Section 5. The script reproduces the example of Section 7 step by step:

- Constructs the first companion pencil for a quadratic scalar polynomial.
- Builds the tangent matrix \(T\) and the constraint matrix \(T_1\).
- Computes the optimal subspace \(S_t\) via SVD (Lemma 3.2).
- Constructs bases for \(\mathcal{GS}\) and \(\mathcal{V}_s\).
- Iteratively solves the recurrence formulas (31)–(32) to obtain first‑ and second‑order derivatives.
- Assembles the versal deformation, including the parameter functions \(\varphi_k(\vec\varepsilon)\) and the transformation matrices \(U(\vec\varepsilon), V(\vec\varepsilon)\).

The code is fully commented and produces the numerical values displayed in the paper.

## 2. Code described in formulas – `latexcodematlab.pdf`

A LaTeX document that restates the key algorithmic steps **using only mathematical formulas** (no executable code). This file serves as a bridge between the theoretical presentation in the manuscript and the actual MATLAB implementation. It includes:

- The exact linear systems for \(D^h\varphi\) and \(D^h\mu\) (equations (31) and (32)).
- The construction of matrices \(L\) and \(K\) from inner products.
- The orthonormalisation procedure that enforces \(K = I\).
- A compact pseudocode summary of the iterative scheme.

This document is meant for readers who wish to re‑implement the algorithm in another language or to understand the algebraic details without parsing MATLAB syntax.

## 3. Outputs in PDF – `versal_deformation_output.pdf`

A compiled PDF containing all numerical results generated by the MATLAB code. It includes:

- The bases of \(S_t\), \(\mathcal{GS}\), and \(\mathcal{V}_s\).
- The matrices \(L\) and \(K\).
- The first‑order derivatives \(A\) and \(A^\mu\).
- The second‑order terms \(B^k\) and \(B^{\mu k}\).
- The final expressions for the versal deformation (parameter functions and transformation matrices).

All numbers are reported with sufficient precision to reproduce the example exactly and to compare directly with the tables in Section 7 of the manuscript.

## How to use these materials

1. Run `verssalALPcodematlab.m` in MATLAB (or GNU Octave) to generate the numerical results.
2. Open `versal_deformation_output.pdf` to see the formatted output tables.
3. Read `latexcodematlab.pdf` for a formula‑only description of the method.

## Relation to the manuscript

- Section 7 (An illustrative example) directly refers to the matrices and values produced by the MATLAB code and shown in `versal_deformation_output.pdf`.
- Appendix A provides the basis construction for \(\mathcal{GS}\), which is implemented in the script.
- Lemma 3.2 and Theorem 3.2 are the theoretical foundations for the SVD‑based steps.
